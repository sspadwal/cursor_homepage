## üé® Design & Style

### The Fonts I Used

- **Main Font**: I used `CursorGothic` to keep it authentic. If the browser doesn't have it, it falls back to standard clean fonts like Helvetica or Arial.

### The Colors I Used

I kept the "Dark Mode" vibe very strict:

- **Background**: A very dark black/yellowish tint (`#14120b`).
- **Cards/Sections**: A slightly lighter dark shade (`#1d1b15`) to make them pop.
- **Main Text**: A soft white (`#edecec`) so it's easy on the eyes.
- **Accent Color**: That signature orange/red (`#f54e00`) for links and hover stuff.
